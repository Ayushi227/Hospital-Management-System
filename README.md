# 🏥 Hospital Management System

A basic web application for managing hospital information like patients, doctors, and appointments. Built with core web technologies to demonstrate CRUD functionality and simple UI interaction.

---

## 🚀 Features

- Add and view patient records
- Add and view doctor details
- Schedule and view appointments

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📌 How to Use

1. Clone the repository  
2. Open `index.html` in a browser  
3. Interact with the UI to add/view records
